# MDX株式会社 コーポレートサイト 公開・更新後チェックリスト

公開時および更新後に確認する項目です。
チェックボックスをコピーして使用してください。

---

## 基本表示確認

- [ ] `https://mdx-inc.co.jp` が HTTPS で正常に開く
- [ ] HTTP（`http://mdx-inc.co.jp`）にアクセスすると HTTPS にリダイレクトされる
- [ ] ページ全体の読み込みが完了し、白画面やエラーが出ない
- [ ] Google Fonts（Noto Sans JP）が適用されている

## PC表示確認

- [ ] Hero セクションのキャッチコピーとボタンが正常表示される
- [ ] 会社概要・主要事業・MDXの強みセクションが崩れていない
- [ ] 会社情報テーブルの内容が最新（代表者・所在地・設立・資本金・メール）
- [ ] `info@mdx-inc.co.jp` のメールリンクをクリックするとメーラーが起動する

## スマホ表示確認

- [ ] ハンバーガーメニューが表示され、タップで開閉できる
- [ ] メニューから各セクションに遷移できる
- [ ] テキストがはみ出したり重なったりしていない
- [ ] ボタン・入力欄が指でタップしやすいサイズになっている
- [ ] 横スクロールが発生していない

## ヘッダー・フッター導線確認

- [ ] ヘッダーのナビリンクが全セクションに正しく遷移する
- [ ] フッターから `privacy.html` に遷移できる
- [ ] フッターから `disclaimer.html` に遷移できる
- [ ] フッターから `careers.html` に遷移できる
- [ ] フッターの著作権年が現在の年になっている

## お問い合わせフォーム確認

- [ ] 未入力で送信ボタンを押すと、必須項目にエラーメッセージが出る
- [ ] メールアドレス欄に不正な形式を入力するとエラーが出る
- [ ] 全項目を正しく入力して送信すると「送信しています…」と表示される
- [ ] 送信後「お問い合わせを受け付けました」の成功メッセージが出る
- [ ] 送信中にボタンが無効化され、二重送信できない
- [ ] Formspree ダッシュボードの Submissions に送信内容が記録されている
- [ ] `info@mdx-inc.co.jp` にメール通知が届く

## OGP・favicon 確認

- [ ] ブラウザタブに favicon が表示される
- [ ] `https://www.opengraph.xyz` で `https://mdx-inc.co.jp` を検索し、OGP画像が表示される
- [ ] OGP のタイトル・説明文が正しく表示される
- [ ] `https://mdx-inc.co.jp/assets/ogp/ogp.png` に直接アクセスして画像が表示される

## 法務ページ・sitemap 確認

- [ ] `https://mdx-inc.co.jp/privacy.html` が正常に表示される
- [ ] `https://mdx-inc.co.jp/disclaimer.html` が正常に表示される
- [ ] 両ページがスマホでも読みやすいレイアウトになっている
- [ ] 最終更新日が正しい日付になっている
- [ ] `https://mdx-inc.co.jp/sitemap.xml` がブラウザで開ける
- [ ] `https://mdx-inc.co.jp/robots.txt` がブラウザで開ける

## 採用ページ確認

- [ ] `https://mdx-inc.co.jp/careers.html` が正常に表示される
- [ ] 応募フォームボタンから Google フォームが開く
- [ ] スマホでも募集要項テーブルが読みやすい
- [ ] 医療・クリニック・患者などの表現が含まれていない
