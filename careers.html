AIを使って、現場の「当たり前」を
変えていく仕事。

IT・業務改善人材 募集中｜正社員

MDX株式会社は、SCS事業・ポータルサイト運営・IT/AI活用支援・業務改善を手がけるBtoB企業です。

私たちが求めているのは、コードを書く人ではありません。現場の業務を理解し、AIツールを使って「仕組みそのもの」を改善できる人です。Claude CodeやGeminiなどのAIツールを日常的に活用し、業務フローの整理・標準化・自動化を推進します。

この仕事が面白い理由

一般的なIT職と異なり、ただ指示された仕様を実装する仕事ではありません。「なぜこの業務が必要なのか」「本当にこの手順でいいのか」から考え、AIの力を借りて業務そのものを設計し直す仕事です。

Claude CodeやGeminiといったAIツールは、使い方次第で生産性を大きく変えます。これらを実務で使いこなしながら、現場にとって本当に意味のある改善を積み重ねていく――それがこのポジションの面白さです。

業務内容

AIツールを活用した業務改善が中心です。「作業をこなす」のではなく「仕組みをつくる」ことに重点を置いています。

  • AIツール（Claude Code / Gemini 等）を活用した業務改善・効率化
  • 業務フローの整理・可視化・標準化
  • 業務マニュアル・手順書の作成・改善
  • データの整理・構造化・活用基盤の構築
  • ITツールの選定・導入・運用サポート
  • 社内の業務課題のヒアリングと改善提案

Claude Code 未経験でも大丈夫な理由

社内にClaude Codeの業務活用マニュアルを整備しています。基本操作から実務での使い方まで、段階的に習得できる環境です。

重要なのはツールの経験ではなく、「業務をどう改善するか」を考える力です。IT業界で実務を経験した方であれば、AIツールの操作は入社後に十分キャッチアップできます。

私たちが一緒に働きたいのは、
「言われたことをやる人」ではなく、
「こうした方がいい」と提案できる人です。

AIは道具です。それを使って何を変えるか――その判断ができる方をお待ちしています。
